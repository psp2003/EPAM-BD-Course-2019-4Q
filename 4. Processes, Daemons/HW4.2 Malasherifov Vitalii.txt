systemd
1. Write two daemons: one should be a simple daemon and do sleep 10 after a start and then do echo 1 > /tmp/homework, the second one should be oneshot and do echo 2 > /tmp/homework without any sleep
	First daemon:
		Create file ~/1d.sh
			#!/bin/bash
			sleep 10
			echo 1 > /tmp/homework
		Create file /etc/systemd/system/1d.service:
			[Unit]
			Description=First Daemon
			[Service]
			ExecStart=/home/vit/1d.sh			
	Second daemon:
		Create file ~/2d.sh
			#!/bin/bash
			echo 2 > /tmp/homework
		Create file /etc/systemd/system/2d.service
			[Unit]
			Description=Second Daemon
			[Service]
			Type=oneshot
			ExecStart=/home/vit/2d.sh
	Check rights of .sh files:
		cd /home/vit
		ll | grep .sh
	Change rights (if non-executable)
		chmod -c 777 .sh
2. Make the second depended on the first one (should start only after the first)
	Change file /etc/systemd/system/2d.service
			[Unit]
			Description=Second Daemon
			Requires=1d.service
			After=1d.service
			[Service]
			Type=oneshot
			ExecStart=/home/vit/2d.sh
3. Write a timer for the second one and configure it to run on 01.01.2019 at 00:00
	Create timer file: /etc/systemd/system/2d.timer
		[Unit]
		Description=Second daemon timer
		[Timer]
		OnCalendar=2019-01-01 00:00
		Persistent=true

4. Start all daemons and timer, check their statuses, timer list and /tmp/homework
